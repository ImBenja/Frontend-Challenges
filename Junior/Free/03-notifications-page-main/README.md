# 📝 Descripción

> Este proyecto es una Pagina de Notificaciones de un club de Ajedrez donde se muestra todo lo que mandan las personas en el club, todo se muestra de manera interactiva. Está diseñada para ser completamente responsive, con diseños adaptados tanto para móviles como para escritorio.

## ✨ Características principales

- **Diseño responsive:** Compatible con móviles y escritorio.

- **Diseño Flexbox:** Optimizado para una visualización perfecta en desktop y mobile.

- **Uso de JavaScript:** Para alternar el estado de las notificaciones.

- **Visualización de la Pagina:** Cuenta con el perfil de cada persona, la notificacion de lo que hizo, aparece si la notificacion es reciente, un boton para leer todas las notificaciones y redes sociales.

## 🛠️ Tecnologías utilizadas

- **_HTML:_** Estructura semántica de las notificaciones.

- **_CSS:_** Estilos con Flexbox.

- **_JavaScript:_** Para leer todas las notificaciones recientes.

- **_Google Fonts:_** Fuente _Plus Jakarta Sans_ para un diseño moderno.

- **_Git:_** Control de versiones.

## 🚀 Instalación

Sigue estos sencillos pasos para ejecutar el proyecto en tu entorno local:

1. **Clona este repositorio**:
   Abre tu terminal y ejecuta el siguiente comando:

   ```bash
   git clone https://github.com/ImBenja/Frontend-Challenges.git

   ```

2. **Navega al directorio del proyecto**:

   ```bash
   cd Frontend-Challenges/Junior/Free/03-notifications-page-main

   ```

3. **Abre el archivo index.html en tu navegador**:
   Puedes hacerlo doble clic en el archivo o arrastrarlo a tu navegador.
